# PENS Room Booking - Backend API

## Description
Backend REST API for managing customer data in the PENS Room Booking System.
Built using ASP.NET Core 8 and Entity Framework Core with SQLite.

## Features
- Customer CRUD
- Pagination & search
- Soft delete
- DTO validation
- Database migration & seeding
- Swagger documentation

## Tech Stack
- ASP.NET Core 8
- Entity Framework Core 8
- SQLite
- Swagger
